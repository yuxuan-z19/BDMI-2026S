```bash

```

```python
import openai

client = openai.OpenAI(api_key="<$API_KEY>", base_url="https://llmapi.paratera.com/v1/")

response = client.chat.completions.create(
    model="模型ID",  # ? 下表中的模型 ID
    messages=[{"role": "user", "content": "Hello world"}],
    max_tokens=32268,  # ? 目前为模型的最大输出长度
)

print(response.choices[0].message.content)
```

> 每个模型的 `max_tokens` （最大输出长度）不同，建议先用简单任务并设置极大值、通过平台返回的日志信息进行确认。
