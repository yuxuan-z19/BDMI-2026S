### PyTorch ABC — Complete Training Example
```python
class Model(torch.nn.Module):
    def __init__(self):
        super().__init__()
        self.linear = torch.nn.Linear(1, 1)

    def forward(self, x):
        return F.sigmoid(self.linear(x))

model = Model()
criterion = torch.nn.BCELoss()
optimizer = torch.optim.SGD(model.parameters(), lr=0.01)

for epoch in range(1000):
    y_pred = model(x_data)           # Forward pass
    loss = criterion(y_pred, y_data) # Compute loss
    optimizer.zero_grad()            # Zero gradients
    loss.backward()                  # Backpropagation
    optimizer.step()                 # Update weights
```

## 3. Convolutional Neural Networks (CNN)

### Multi-layer Network Topologies
- **Feedforward**: MLP, CNN — information flows forward only
- **Feedback**: RNN — outputs loop back as inputs
- CNN is a feedforward network with **local (sparse) connections**, unlike MLP's full connections

### Convolution Operation
- A **convolution kernel** (filter) is a small multi-dimensional array
- The convolution operation applies the kernel as a sliding window over the input, computing weighted sums
- Results in fewer parameters than fully connected networks

### CNN Architecture Components
| Layer | Function |
|-------|----------|
| Convolutional Layer | Feature extraction via kernels |
| Activation (ReLU/Sigmoid) | Nonlinear transformation |
| Pooling Layer | Downsampling (max/avg pool) |
| Normalization Layer | Stabilize training |
| Dropout Layer | Regularization (prevent overfitting) |

### CNN Advantages
- Fewer parameters than FCN
- Parallelizable computation
- Low inference latency

### Historical CNN Architectures
- **LeNet-5** (1989): Handwritten digit recognition
- **AlexNet** (2012): Won ImageNet, triggered deep learning revolution
  - 8 layers, 60M parameters, GPU training, dropout regularization

## 4. Transformer Overview (Introduction)

### Key Terminology
| Term | Description |
|------|-------------|
| Self-attention | Relates different positions within a single sequence |
| Scaled dot-product attention | Core attention computation |
| Positional encoding | Injects sequence order information |
| Multi-head attention | Parallel attention with different learned projections |
| Masking | Prevents decoder from seeing future tokens |

### Transformer Structure
- **Encoder-Decoder** architecture (Seq2Seq)
- **Encoder**: Multi-head self-attention + Feed-forward network
- **Decoder**: Masked self-attention + Encoder-decoder attention + Feed-forward network

### Transformer Family (introduced)
| Model | Type | Strength |
|-------|------|----------|
| BERT | Encoder-only | Language understanding |
| GPT | Decoder-only | Language generation |
| T5/BART | Encoder-decoder | Seq2seq tasks |
