# Week 8 Study Memo — Computer Vision (CV)

## 1. Human Visual Perception

- Approximately **70% of human information** comes from vision.
- The visual processing pipeline: Retina → Optic nerve → Visual cortex (V1 → V2 → V4 → IT)
- **Rod cells**: Responsible for dim-light (scotopic) vision
- **Cone cells**: Responsible for color (photopic) vision (~6-7M cones vs. ~125M rods)
- The hierarchical structure of the visual cortex **inspired deep learning** for computer vision.

---

## 2. Computer Vision Tasks

| Task | Description |
|------|-------------|
| **Image Classification** | Assign a label to an image |
| **Object Localization** | Find where an object is (bounding box) |
| **Object Detection** | Classify + localize multiple objects |
| **Semantic Segmentation** | Label every pixel in an image |
| **Instance Segmentation** | Segment + identify individual objects |

---

## 3. Traditional CV Methods

### Classical Techniques
- **OpenCV**: Powerful image processing library (`pip install opencv-python`)
- **Scikit-image**: Image processing toolkit
- Feature detection: SIFT, SURF, Harris corner, FAST, BRIEF, BRISK
- Segmentation: Watershed algorithm, Delaunay triangulation

### Traditional ML for CV
- SVM, Naïve Bayes, Decision Trees for classification
- **HOG** (Histogram of Oriented Gradients) for object recognition
- **Haar classifiers** for face detection (used in OpenCV)
- **Boosting**: Ensemble methods (AdaBoost, Random Forest) combining weak classifiers into strong ones

---

## 4. Deep Learning for Image Classification

### ImageNet & ILSVRC
- **ImageNet**: 15M+ images, ~22K categories (created by Prof. Fei-Fei Li, Stanford)
- **ILSVRC**: Competition using 1K-class subset (1.2M training, 50K validation, 150K test images)

### CNN Evolution
| Network | Year | Key Innovation |
|---------|------|---------------|
| **LeNet-5** | 1989 | First successful CNN (handwriting) |
| **AlexNet** | 2012 | Deep CNN + GPU + Dropout → Won ImageNet |
| **NiN** | 2013 | 1×1 convolutions, global pooling |
| **VGG** | 2014 | 3×3 kernels stacked deeply |
| **GoogLeNet** | 2014 | Inception modules |
| **ResNet** | 2015 | Skip connections / residual learning |
| **DenseNet** | 2017 | Dense block connections |

### AlexNet's Breakthrough (2012)
- 8 layers, 60M parameters vs. previous 3-layer/1M-parameter models
- **GPU training** for the first time
- 1.2M training images (vs. thousands previously)
- **Dropout** regularization
- Triggered the modern deep learning revolution

## 5. Vision Transformer (ViT)

### Concept
- Splits images into fixed-size **patches** (e.g., 16×16).
- Each patch is linearly embedded and treated as a token.
- Position embeddings are added, then fed to a standard Transformer encoder.
- A learnable `[CLS]` token is used for classification.

### ViT Variants
| Variant | Innovation |
|---------|-----------|
| **DeiT** | Knowledge distillation for data efficiency |
| **Swin Transformer** | Hierarchical + shifted windows for scalability |
| **LeViT** | Multi-level processing for efficiency |
| **XciT** | Cross-covariance attention (linear scaling) |

### MLP-Mixer
- Alternative: uses **only MLPs** (no attention or convolution).
- Token-mixing MLP + Channel-mixing MLP.

## 6. Object Detection

### R-CNN Family
| Model | Approach |
|-------|----------|
| R-CNN | Region proposals → CNN features → SVM |
| Fast R-CNN | Shared CNN features + ROI pooling |
| Faster R-CNN | Region Proposal Network (RPN) |

### One-Stage Detectors
- **YOLO** (You Only Look Once): Single forward pass for detection
- **SSD** (Single Shot Detector): Multi-scale feature maps
- **DETR**: Transformer-based detection (end-to-end)

---

## 7. Image Segmentation
- **Mask R-CNN**: Extends Faster R-CNN with pixel-level masks
- **TensorMask**: Dense mask prediction
- Transformer-based segmentation methods

This week provided a comprehensive tour of computer vision — from biological visual systems to modern deep learning approaches. I learned the evolution of CNN architectures from LeNet to ResNet, the paradigm shift introduced by Vision Transformers (ViT), and the hierarchy of CV tasks (classification → detection → segmentation).
