## 2. Transformer Recap & Why It Dominates

### Advantages
- **Parallel computation**: Processes entire sequences simultaneously (unlike RNN)
- **Long-range dependencies**: Attention directly connects any two positions
- **Multi-Head Attention**: Each head learns different features
- **Scalable**: Stack more layers for more capacity

### Limitation
- No inherent notion of sequence order → requires **positional encoding**

---

## 3. Large Language Models (LLMs)

### GPT Family (Generative Pre-trained Transformer)
- GPT is a **decoder-only** Transformer for language generation.
- Progression: GPT → GPT-2 → GPT-3 → GPT-4
- Key capabilities: Zero-shot, One-shot, Few-shot learning, Fine-tuning

### DeepSeek V3
- Open-source large language model with key innovations:
  - **MLA (Multi-head Latent Attention)**: Low-rank KV compression for efficient inference
  - **MoE (Mixture of Experts)**: Sparse activation for efficient scaling
  - **Multi-token prediction**: Predict multiple tokens simultaneously

### Transformer Life Tree
| Type | Models | Characteristic |
|------|--------|---------------|
| **GPT-like** (Autoregressive) | GPT-1/2/3/4, LLaMA | Left-to-right generation |
| **BERT-like** (Autoencoding) | BERT, RoBERTa | Bidirectional understanding |
| **BART/T5-like** (Seq2Seq) | T5, BART | Encoder-decoder for translation/summarization |

### GLM (General Language Model)
- Developed by Zhipu AI / Tsinghua University
- GLM-4.5: New paradigm for native integration of large models

---

## 4. Multimodal AI

### Vision Transformer (ViT)
- Splits images into patches, treats them as tokens for Transformer processing
- `[CLS]` token for classification

### CLIP (Contrastive Language-Image Pre-training)
- Aligns image and text in a shared vector space via contrastive learning
- Enables zero-shot image classification by comparing image embeddings with text descriptions

### Foundation Models
- A single base model (often LLM-based) adapted for multiple modalities: text, image, audio, video

## 5. AI Ethics & Challenges

### Positive Impacts of AI
- Healthcare, scientific research, productivity, accessibility, creative tools

### Problems & Risks
| Issue | Description |
|-------|-------------|
| **Job displacement** | AI replacing call centers, security, logistics workers, drivers |
| **Decision failures** | Self-driving accidents, biased chatbot outputs |
| **Fairness** | Training data biases leading to discriminatory models |
| **Interpretability** | Deep learning models as "black boxes" |
| **Privacy** | Models trained on sensitive data |
| **Data misuse** | Biased policing data leading to biased AI systems |

### Notable Examples
- **Word2Vec bias**: `doctor - man + woman = nurse` reflects societal bias in training data
- **Survivorship bias**: Parachute evaluation problem, WWII airplane armor problem
- **Data discrimination**: "Big data price discrimination" (different prices on iPhone vs. Android)

### Solutions: Responsible AI
- **Interpretability**: Make AI decisions explainable
- **Privacy**: Data protection measures (GDPR, China's Personal Information Protection Law)
- **Fairness**: Audit training data and model outputs for bias
- **Security**: Identify and mitigate potential threats

### Famous Quotes
> *"As soon as it works, no one calls it AI anymore."* — **John McCarthy**

> *"We tend to overestimate the effect of a technology in the short run and underestimate the effect in the long run."* — **Amara's Law**
