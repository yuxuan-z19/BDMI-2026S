# Week 5 Study Memo — Multi-layer Networks, Automatic Differentiation & FCN

## 1. Multi-layer Neural Networks (Deep Neural Networks)

### From Single Neurons to Deep Networks
- A single neuron performs **linear classification**.
- Multiple neurons combined into layers can solve **complex nonlinear** classification problems.
- Multi-layer networks are also called **deep neural networks (DNN)**.

### Network Topology
| Type | Description | Example |
|------|-------------|---------|
| Feedforward | Information flows in one direction | MLP, CNN |
| Feedback | Outputs feed back as inputs | RNN |
| Memory Network | Maintains external memory | DNC |

---

## 2. Fully Connected Networks (FCN / MLP)

### Structure
- Also known as **Multi-Layer Perceptron (MLP)** or **Dense Networks**.
- Every neuron in one layer connects to every neuron in the next layer.
- Components: **Input layer → Hidden layer(s) → Output layer**.

### Key Properties
- **Dense connections**: All-to-all connectivity between adjacent layers
- Parameters: weight matrices $W$ and bias vectors $b$
- Forward computation: $h = f(Wx + b)$

---

## 3. Automatic Differentiation (AutoDiff)

### Concept
- AutoDiff computes **exact gradients** of functions by applying the chain rule systematically.
- Essential for training neural networks via **backpropagation**.

### Comparison
| Method | Precision | Speed |
|--------|-----------|-------|
| Numerical differentiation | Approximate | Slow |
| Symbolic differentiation | Exact | Can be complex |
| **Automatic differentiation** | **Exact** | **Efficient** |

### Backpropagation
- Computes gradients of the loss function with respect to all weights.
- Uses the **computational graph** to chain partial derivatives.
- Flow: Forward pass (compute loss) → Backward pass (compute gradients).

---

## 4. PyTorch Autograd

### How It Works
```python
import torch

x = torch.tensor(2.0, requires_grad=True)
y = x ** 2 + 3 * x
y.backward()
print(x.grad)  # dy/dx = 2x + 3 = 7.0
```

### Key Concepts
- `requires_grad=True`: Tells PyTorch to track operations for gradient computation
- `.backward()`: Computes gradients via backpropagation
- `.grad`: Stores the computed gradient

## 5. PyTorch Layers & Modules

### Building Blocks
```python
import torch.nn as nn

# Linear layer
layer = nn.Linear(in_features=10, out_features=5)

# Sequential model
model = nn.Sequential(
    nn.Linear(784, 256),
    nn.ReLU(),
    nn.Linear(256, 10)
)
```

### Custom Model
```python
class MyModel(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(784, 256)
        self.fc2 = nn.Linear(256, 10)
        self.relu = nn.ReLU()

    def forward(self, x):
        x = self.relu(self.fc1(x))
        return self.fc2(x)
```
