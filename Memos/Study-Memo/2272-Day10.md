## 深度学习概述
### 线性分类器
- 图像分类：通过线性分类器（如 SVM）将图像映射到类别
- 损失函数：衡量预测与真实标签的差距（如多项式逻辑回归）
- 优化：通过梯度下降最小化损失
- 反向传播：利用计算图（算子组成），链式法则计算梯度并更新参数
- 局限：线性分类器对复杂模式表达能力不足
### 深度人工神经网络基础
- 人工神经元：模仿生物神经元，通过激活函数引入非线性
- 多层全连接网络（MLP/Dense）：最基本的深度网络结构
- 卷积网络（CNN）：局部连接、权重共享，擅长图像处理（如 LeNet-5、AlexNet）
- 循环网络（RNN）：反馈连接，适合序列数据
- 网络拓扑：前馈网络（MLP/CNN）vs. 反馈网络（RNN）
- 里程碑 - AlexNet (2012)：8 层、6 千万参数、GPU 训练，带动深度学习热潮
- ImageNet：李飞飞团队建立，1500 万张图片，22000 类，ILSVRC 挑战赛的基石
### 训练深度网络的关键点
- 数据预处理、网络结构与损失函数设计
- 梯度下降 + 学习率调度（指数衰减、分步衰减等）
- 优化器：RMSProp、Adam
- 正则化防止过拟合：L2 正则、数据增强、Dropout、集成学习
### Transformer
- Encoder-Decoder 结构（Seq2Seq），可并行训练
- 核心：自注意力机制
  - Scaled Dot-Product Attention（Q、K、V）
  - 多头注意力（Multi-Head Attention）
  - 位置编码（Positional Encoding）
  - 遮挡（Masking）：填充遮挡 + 前瞻遮挡
- 残差连接 + 层归一化：`LayerNorm(x + Sublayer(x))`
- 发展分支：
  - BERT（双向编码器）— 语言理解
  - GPT（自回归）— 语言生成
  - BART/T5（Seq2Seq）
- 应用：ViT（视觉）、iTransformer（时间序列预测）

## 深度学习语音识别
### 语音识别基础
- ASR：自动将语音信号转换为文字
- 语音信号表示：波形图 → FFT 频域图 → STFFT 时频谱（保留时间+频率信息）
- 评价指标：词错误率（WER）= (替换+插入+删除) / 标准答案长度，可超过 100%
- 核心挑战：口音、语气、噪声、鸡尾酒会问题（多人同时说话）
### 传统方法
- 基本框架：声学模型 + 语言模型（贝叶斯公式）
- 处理流水线：STFT → MFCC → 音素序列 → 字母序列 → 词汇序列
- 经典技术：HMM、GMM、维特比搜索、n-gram 语言模型
### 深度学习引入（2006 年后）
- 混合方法：DNN 与 HMM 等传统模块混合
- 端到端方法：直接用神经网络学习 F(h,x) 函数变换，无需传统流水线
- 效果：WER 下降 30%+，突破实用临界点
- 端到端流水线：特征提取 → 声学模型（NN）→ 语言模型 → 转录
### CTC（Connectionist Temporal Classification）
- 解决的问题：输入语音 X 和输出文字 Y 长度不同且无明确对齐
- 核心思想：对齐无关（alignment-free），对给定 X 计算所有可能 Y 的概率分布
- CTC 折叠：合并重复字符，用 blank 分隔
- CTC Loss：基于动态规划高效计算（避免枚举所有对齐路径）
- 推断：贪心解码 / 束搜索（Beam Search）
### DeepSpeech2
- 架构：卷积层 + 循环层（RNN/GRU）+ 全连接层
- 输入：时频谱图；输出：英文字母或中文字符
- 优化技巧：BatchNorm、SortaGrid（短句优先）、Lookahead Convolution（替代双向 LSTM 降低时延）
- 开源实现：Mozilla DeepSpeech、PaddleSpeech

## Whisper
### 核心特点
- OpenAI 开发，68 万小时多语言（98 种）监督数据训练
- 使用 Transformer Seq2Seq 架构
- 音频切分为 30 秒段，转 log-Mel 频谱图 → Encoder → Decoder
- 多任务训练：语音识别、语音翻译、语音活动检测统一为同一模型
- 特殊标记（special tokens）区分不同任务

### 训练细节
- FP16 混合精度 + 动态损失缩放 + 激活检查点
- AdamW 优化器 + 梯度范数裁剪
- 学习率预热 2048 步后线性衰减至零
- batch size 256，仅 2-3 个 epoch（数据量大，过拟合风险低）
- 无数据增强、无显式正则化，依赖数据多样性保证泛化

### 模型规模与使用
- 五种尺寸（tiny / base / small / medium / large），部分仅英文
- 安装：`pip install whisper`（兼容 Python 3.8-3.10 + PyTorch）
- 可通过命令行、Python API、HuggingFace/Transformers.js（浏览器端）调用

## Torchaudio

### 核心功能
- 加载/保存：load()、save()，支持 WAV/FLAC/MP3/OGG
- 信号处理：resample()、fade()、volumne()
- 频谱变换：spectrogram()、griffinlim()、mfcc()
- 数据集：内置常用数据集，支持自定义加载

### 预处理流程
加载音频 → 重采样（统一16kHz）→ 降噪（频谱减法）→ 分帧加窗（汉明窗）→ 提取梅尔频谱图（2D特征）

### 模型架构
```
输入: 梅尔频谱图 (batch, 1, freq, time)
→ Conv2d(1,32,3x3) + ReLU + MaxPool
→ Conv2d(32,64,3x3) + ReLU + MaxPool
→ Flatten
→ Linear(64*16*16, 128) + ReLU + Dropout(0.5)
→ Linear(128, num_classes)
→ 输出: 类别概率
```
```python
class SimpleAudioClassifier(nn.Module):
    def __init__(self, num_classes):
        super().__init__()
        self.conv1 = nn.Conv2d(1, 32, 3)
        self.conv2 = nn.Conv2d(32, 64, 3)
        self.fc1 = nn.Linear(64*16*16, 128)
        self.fc2 = nn.Linear(128, num_classes)
    
    def forward(self, x):
        x = F.relu(self.conv1(x))
        x = F.max_pool2d(x, 2)
        x = F.relu(self.conv2(x))
        x = F.max_pool2d(x, 2)
        x = x.view(x.size(0), -1)  # Flatten
        x = F.relu(self.fc1(x))
        x = F.dropout(x, 0.5)
        return self.fc2(x)
```
