### 2. 线性分类器
#### 图像分类与 SVM
- 线性分类器（如 SVM）通过学习线性决策边界对图像进行分类。
- 损失函数衡量预测与真实标签的差距，常用交叉熵或合页损失。
- 优化：通过梯度下降等方法最小化损失函数。
- 反向传播：利用链式法则计算梯度，更新参数。

#### 线性分类器的局限性
- 无法处理非线性可分问题（如异或问题）。
- 对复杂模式（如多模态分布）表现不佳。

### 3. 深度人工神经网络基础
#### 人工神经元
- 生物仿生：输入加权求和 → 非线性激活（ReLU、Sigmoid、Tanh 等）。
- 输出：$y = f\left(\sum w_i x_i + b\right)$

#### 多层全连接网络（MLP / Dense）
- 由输入层、多个隐藏层、输出层组成。
- 连接方式：层间全连接（每个神经元与下一层所有神经元相连）。

#### 卷积神经网络（CNN）
- 仿生视觉皮层，利用局部连接和权重共享。
- 核心操作：卷积（卷积核滑动）、池化（降采样）、非线性激活。
- 经典结构：LeNet‑5（手写识别）、AlexNet（2012 ImageNet 冠军，8 层，6000 万参数，GPU + Dropout）。
- ImageNet 数据集：1500 万图像，2.2 万类，为深度学习提供数据基础。

#### 循环神经网络（RNN）
- 处理序列数据，具有隐藏状态的时间反馈连接。
- 常见变种：LSTM、GRU（解决梯度消失/爆炸）。

### 5. Transformer
#### 核心概念
- **自注意力（Self‑Attention）**：序列内每个位置与其他位置计算注意力权重。
- **缩放点积注意力（Scaled Dot‑Product Attention）**
  $$ \text{Attention}(Q,K,V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V $$
  - $Q$：查询，$K$：键，$V$：值，$d_k$：键维度。
- **多头注意力（Multi‑Head Attention）**：多个注意力头并行，捕捉不同子空间特征。
- **位置编码（Positional Encoding）**：为序列注入位置信息（正弦/余弦函数）。
- **遮挡（Masking）**：填充遮挡（忽略 padding）和前瞻遮挡（防止未来信息泄露）。

#### Transformer 结构
- **编码器（Encoder）**：$N$ 层，每层包含多头自注意力 + 前馈网络（FFN），残差连接 + 层归一化。
- **解码器（Decoder）**：$N$ 层，每层包含带遮挡的多头自注意力 + 编码器‑解码器注意力 + FFN。
- **输入/输出嵌入**：词嵌入 + 位置编码。

#### Transformer 应用
- **BERT**：双向编码器表示，适合自然语言理解（分类、问答等）。
- **GPT**：自回归解码器，适合自然语言生成。
- **ViT**：将图像分割为 patch，线性嵌入后送入 Transformer，用于图像分类。
- **iTransformer**：反转 Transformer 用于时间序列预测。

### 2. 评判指标：词错误率（WER）
- 对齐识别结果与标准答案，计算最少替换（S）、插入（I）、删除（D）总数除以标准答案长度：
  $$ WER = \frac{S + I + D}{N} $$
- 示例：标准答案 “my name is Andy Chen”，识别为 “my nick name are Andy Chen” → 替换2处、插入1处 → WER = 3/5 = 60%
- WER 可能超过 100%（如 “recognize speech” → “wreck a nice beach” → 4/2 = 200%）

### 3. 语音识别的传统方法
#### 基本方程
给定语音信号 $X$ 和单词序列 $W$，最优识别结果：
$$ W^* = \arg\max_W P(W \mid X) = \arg\max_W P(X \mid W) \cdot P(W) $$
- $P(X \mid W)$：声学模型
- $P(W)$：语言模型

#### 经典架构
- 特征提取：STFT、MFCC（梅尔频率倒谱系数）
- 声学模型：上下文相关音素模型、HMM、GMM
- 解码器：维特比搜索
- 语言模型：$n$-gram

### 4. 深度学习的引入
#### 混合模型（Alloy）
- DNN 取代 GMM 与 HMM 结合，提升准确率
- 词错误率下降 30% 以上，突破实用临界点

#### 纯深度学习模型
- 用 DNN 逼近函数 $w = F(v, h)$，直接从语音信号映射到文字
- 足够数据 + 计算资源可构建高性能 LVCSR 系统

### 5. 端到端语音识别
#### 流水线
1. 特征提取：原始波形 → 时频谱图
2. 声学模型：特征向量 → 字符/音素概率
3. 语言模型 + 搜索：解码出最可能转录

#### 关键创新：RNN + CTC（Alex Graves 等）
- 解决语音与文字之间长度不同、无对齐的问题

### 6. 连接主义的时间分类（CTC）
#### 解决的问题
- 输入语音 $X$ 长度 $T$，输出文字 $Y$ 长度 $U$，通常 $T \gg U$
- 无预先对齐，需学习对齐方式

#### CTC 原理
- 引入空白标记 $\epsilon$，允许重复字符和空白
- RNN 在每个时间步输出概率分布 $p_t(a_t \mid X)$
- 所有可能路径（对齐）通过 **CTC 折叠**（合并重复并去掉空白）得到输出序列
- 目标函数：最大化条件概率 $P(Y \mid X)$

#### CTC 损失
$$ \mathcal{L} = -\ln P(Y \mid X) $$
- 可通过动态规划（前向‑后向算法）高效计算
- 常用的开源实现：`warp-ctc`、TensorFlow 的 `tf.nn.ctc_loss`

#### CTC 推断
- **贪心解码**：每时刻取最大概率，合并重复、去除空白
- **束搜索（Beam Search）**：保留多条候选路径，考虑概率之和，效果更好

## 3. Whisper 自动语音识别系统
### 1. 概述
- **Whisper**：OpenAI 开发的自动语音识别（ASR）系统
- 训练数据：从网络收集的 **68 万小时** 多语言（98 种语言）和多任务监督数据
- 识别能力：OpenAI 宣称达到人类水准

### 2. 核心技术
#### 模型架构
- **Transformer Seq2Seq 模型**（编码器‑解码器）
- 输入处理：
  1. 音频分割为 30 秒小段
  2. 转换为 **log‑Mel 频谱图**
  3. 传入编码器
- 解码器输出：文字说明 + 特殊标记（指导语言识别、时间戳、多语言转录、语音翻译等）

#### 多任务训练格式
- 联合表示为一个令牌序列，单个模型替代传统语音处理管道的多个阶段
- 任务包括：多语言语音识别、语音翻译、语音活动检测等
- 使用特殊标记作为任务描述符或分类目标

### 3. 训练细节
- **并行策略**：跨加速器数据并行 + FP16 混合精度 + 动态损失缩放 + 激活检查点
- **优化器**：AdamW + 梯度范数裁剪
- **学习率调度**：前 2048 次更新预热，之后线性衰减至零
- **批大小**：256 个样本段
- **总更新次数**：220 次（约 2‑3 个数据周期）
- **过拟合**：周期短，风险低
- **正则化**：无数据增强、无显式正则化，泛化依赖数据集内在多样性
- **环境**：Python 3.9.9 + PyTorch 1.10.1（兼容 3.8‑3.10 及新版 PyTorch）

## 4. PyTorch 音频识别实战
### 1. 音频识别概述
- **核心定义**：用深度学习模型对音频信号进行分析，识别语义内容
- **技术本质**：将时域信号转换为特征表示，通过神经网络分类
- **典型场景**：语音助手、声纹识别、音乐分类、声音事件检测

### 2. torchaudio：PyTorch 音频处理库
- 安装：`pip install torchaudio`
- **音频加载与保存**：`load()`（支持 WAV, FLAC, MP3, OGG）、`save()`
- **信号处理**：`resample()`（重采样）、`fade()`（淡入淡出）、`volume()`（音量）、`extrapolate()`（补齐）
- **频谱变换**：`spectrogram()`（频谱图）、`griffinlim()`（频谱转波形）、`mfcc()`（梅尔频率倒谱系数）
- **数据集**：内置常用音频数据集，与 torchvision 风格统一

### 3. 音频预处理：从波形到频谱
| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 加载音频 | `torchaudio.load()`，获取波形、采样率、通道数 |
| 2 | 重采样 | 统一到目标采样率（常用 16 kHz 或 22.05 kHz） |
| 3 | 降噪处理 | 频谱减法等，提升信噪比 |
| 4 | 分帧加窗 | 汉明窗或汉宁窗，保留时域局部特征 |
| 5 | 提取特征 | 计算梅尔频谱图、MFCC 或梅尔滤波器组，转为 2D 特征 |

- **输出**：可供 CNN 处理的频谱图（形状：`(batch, 1, freq, time)`）

### 4. 简单音频分类模型架构

输入：梅尔频谱图 (batch, 1, 64, 64)

↓ Conv2d(1, 32, 3×3) + ReLU + MaxPool2d

↓ Conv2d(32, 64, 3×3) + ReLU + MaxPool2d

↓ Flatten

↓ Linear(64×16×16, 128) + ReLU + Dropout(0.5)

↓ Linear(128, num_classes)

输出：各类别概率（softmax）

- 可替换为 ResNet 等预训练模型提升效果

### 5. 训练代码实战
#### 训练三要素
```python
criterion = nn.CrossEntropyLoss()           # 损失函数
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)  # 优化器

#### 训练循环
```python
for epoch in range(num_epochs):
    model.train()
    for batch_idx, (wavs, labels) in enumerate(train_loader):
        optimizer.zero_grad()
        outputs = model(wavs)
        loss = criterion(outputs, labels)
        loss.backward()
        optimizer.step()
    print(f"Epoch {epoch+1}: Loss={loss:.4f}")
```

#### 训练技巧
- 学习率调度器

### 6.推理与预测
```python
model.eval()
