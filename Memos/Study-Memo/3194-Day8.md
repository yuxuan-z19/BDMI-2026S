## 1. 循环网络及其变种
### 1. 核心术语
- 循环网络（Recurrent Neural Network, RNN）
- 基本循环网络（Vanilla RNN）
- 通过时间反向传播（Back‑Propagation Through Time, BPTT）
- 梯度截断（Gradient Clipping）
- 长短时记忆网络（Long Short‑Term Memory, LSTM）
- 门结构（Gate）
- 门控循环单元（Gated Recurrent Unit, GRU）

### 2. 基本循环网络（Vanilla RNN）
#### 结构特点
- 引入反馈连接：当前时刻的隐藏状态传递到下一时刻
- 权重共享：在所有时间步上使用相同的权重矩阵
  - $U$（或 $W_{xh}$）：输入层到隐藏层
  - $V$（或 $W_{hy}$）：隐藏层到输出层
  - $W$（或 $W_{hh}$）：隐藏层到隐藏层
- 偏置：$b$（隐藏层），$c$（输出层）

#### 前向计算方程
$$ h_t = \tanh(U x_t + W h_{t-1} + b) $$
$$ o_t = V h_t + c $$
$$ \hat{y}_t = \text{softmax}(o_t) $$

#### 深度 RNN 与双向 RNN
- **深度 RNN**：堆叠多个循环层，形成栈式结构
- **双向 RNN**：一个 RNN 从前向后，另一个从后向前，输出同时依赖过去和未来
  - 适用于手写识别、语音识别、自然语言处理（需收齐全部输入，时延较大）

### 3. 训练：通过时间反向传播（BPTT）
#### 损失函数
- 总损失为所有时间步损失之和：
  $$ L = \sum_{t=1}^{T} L^{(t)} $$
  其中 $L^{(t)}$ 通常是负对数似然。

#### BPTT 特点
- 将循环网络按时间展开为计算图，执行前向与反向传播
- 时间成本 $O(T)$，内存成本 $O(T)$（需保存各时间步状态）
- 前向传播无法并行化，因为时间步之间固有顺序

#### 梯度问题
- **梯度消失**：远距离时间步的梯度难以传递
- **梯度爆炸**：梯度值过大，使用**梯度截断**解决（如设置阈值，超过则缩放）

### 4. 长短时记忆网络（LSTM）
#### 门结构
- 门：输出 $\sigma(\text{control}) \in [0,1]$，与输入逐元素相乘
- 三个门：
  - **输入门 $i_t$**：控制是否写入记忆单元
  - **遗忘门 $f_t$**：控制是否清除历史记忆
  - **输出门 $o_t$**：控制是否从记忆单元读出

#### 记忆单元 $c_t$
- 核心是记忆单元 $c_t$，作为累加器
- 候选值：$\tilde{c}_t = \tanh(W_c [h_{t-1}, x_t] + b_c)$
- 更新：$c_t = f_t \odot c_{t-1} + i_t \odot \tilde{c}_t$
- 隐藏状态：$h_t = o_t \odot \tanh(c_t)$

#### LSTM 优势
- 解决基本 RNN 收敛慢的问题
- 能够建模长距离依赖（Long‑term dependencies）

### 5. 门控循环单元（GRU）
#### 与 LSTM 的区别
- 合并记忆单元 $c_t$ 和隐藏状态 $h_t$ 为一个状态
- 合并输入门与遗忘门为**更新门** $z_t$
- 只保留两个门：更新门 $z_t$ 和重置门 $r_t$

#### GRU 前向计算
$$ z_t = \sigma(W_z [h_{t-1}, x_t] + b_z) $$
$$ r_t = \sigma(W_r [h_{t-1}, x_t] + b_r) $$
$$ \tilde{h}_t = \tanh(W_h [r_t \odot h_{t-1}, x_t] + b_h) $$
$$ h_t = (1 - z_t) \odot h_{t-1} + z_t \odot \tilde{h}_t $$

#### GRU 优点
- 参数少，计算更快，需要的训练数据更少
- 许多任务中性能与 LSTM 相当
- 若数据量极大，LSTM 表达能力更强

### 6. RNN 的主要应用

| 应用领域 | 具体任务 |
|----------|----------|
| 序列预测 | 手写识别、股价预测、天气预测 |
| 自然语言处理 | 语音识别、文本分类、文本生成、机器翻译 |
| 计算机视觉 | 图片注解（Image Captioning） |
| 其他 | 代码生成、情感分析 |

## 2. 计算机视觉目标检测与评价指标

### 1. 计算机视觉的主要任务
- **图像分类**：判断图像中的物体类别
- **目标定位**：确定物体的位置（边界框）
- **目标检测**：同时完成分类和定位
- **图像分割**：对图像中的每个像素进行场景分析和标注

### 2. 核心评价指标

#### 基本定义
- **精确率（Precision）**：预测为正的样本中，实际为正的比例  
  $$ \text{Precision} = \frac{TP}{TP+FP} $$
- **召回率（Recall）**：实际为正的样本中，被正确预测为正的比例  
  $$ \text{Recall} = \frac{TP}{TP+FN} $$
- **准确率（Accuracy）**：分类正确的样本数占总样本数的比例  
  $$ \text{Accuracy} = \frac{TP+TN}{TP+FN+FP+TN} $$
- **错误率（Error Rate）**：分类错误的样本数占总样本数的比例，$= 1 - \text{Accuracy}$
- **灵敏度（Sensitive）**：同召回率，衡量识别正例的能力
- **特异度（Specificity）**：实际为负的样本中，被正确预测为负的比例  
  $$ \text{Specificity} = \frac{TN}{TN+FP} $$

#### 混淆矩阵（Confusion Matrix）
- 用于可视化分类结果，行表示真实类别，列表示预测类别。

#### 示例计算
假设有100张图像（70猫，30狗），检测系统找到50张猫图，其中40张是真正的猫。
- TP=40, FN=30, FP=10, TN=20
- 精确率 = 40/(40+10) = 80%
- 召回率 = 40/(40+30) = 4/7 ≈ 57.1%
- 准确率 = (40+20)/100 = 60%

### 3. 目标检测专用指标

#### mAP（Mean Average Precision）
- **定义**：PASCAL VOC 2007 数据集使用的评价指标，衡量模型在不同召回率下的平均精确率。
- **计算**：对每个类别，计算 PR 曲线下的面积（AP），再对所有类别取平均得 mAP。
- **含义**：mAP 越高，模型检测精度越好。它平衡了精确率和召回率，选择合适阈值使检测器处于最佳工作状态。
- **F1 分数**：精确率和召回率的调和平均值，同时考虑两者影响。

#### PR 曲线与 AUC
- **PR 曲线**：以召回率为横轴、精确率为纵轴绘制的曲线。
- **AUC（Area Under Curve）**：PR 曲线下的面积。AUC 越大，检测识别精度越高。
- 应用示例：VIVA 2015 交通标志检测竞赛使用 PR 曲线 AUC 作为评价指标。

### 4. 相关竞赛与数据集
- **PASCAL VOC Challenge**：计算机视觉领域的经典挑战，提供多类别标注图像，基于分类精确率、召回率和效率排名。
- **VIVA（Vision for Intelligent Vehicles and Applications）**：智能车辆视觉应用竞赛，使用 PR 曲线 AUC 评估。

## 3. 计算机视觉（一）
### 1. 核心术语
- 人类感知（Human Perception）
- 计算机视觉（Computer Vision）
- 分类（Image Classification）
- 定位（Localization）
- 目标检测（Object Detection）
- 语义分割（Semantic Segmentation）
- 自主/无人驾驶（Self-Driving / Driverless）
- 自动驾驶车（Autonomous Vehicles）

### 2. 人类视觉感知系统
- 人的信息约 70% 来自视觉
- 视网膜：视杆细胞（明暗视觉）+ 视锥细胞（彩色视觉）
- 视觉通路：腹侧通路（Ventral pathway，物体识别）、背侧通路（Dorsal pathway，空间位置）
- 大脑皮层的视觉处理结构（V2→V4→IT）为深度学习提供了灵感

### 3. 计算机视觉的任务
| 任务 | 输出 |
|------|------|
| 分类 | 类别标签 |
| 定位 | 边界框 |
| 检测 | 类别 + 边界框 |
| 语义分割 | 每个像素的类别 |

### 4. 计算机视觉的实现方法
#### 4.1 传统方法
- **OpenCV**：图像处理库（红绿黄圆形识别等）
- **Scikit-image**：RGB‑HSV 颜色转换等
- **传统算法**：分水岭分割、三角剖分（Delaunay）、角点跟踪（Horn‑Schunk, Kanade）
- **局部特征检测**：斑点检测（LoG, DoH, SIFT, SURF）、角点检测（Harris, FAST）、二进制描述子（BRIEF, BRISK）
- **传统机器学习**：SVM、朴素贝叶斯、决策树、HOG、Boosting（AdaBoost）、随机森林、Haar 分类器（人脸识别）

#### 4.2 深度学习方法
- **卷积神经网络（CNN）**：LeNet‑5（手写数字），AlexNet（2012 ImageNet 冠军，8 层，6000 万参数，GPU + Dropout）
- **后续发展**：NiN（1×1 卷积）、VGG（小卷积核堆叠）、GoogLeNet（Inception）、ResNet（残差，Skip Connection）、DenseNet（密集连接）
- **Vision Transformer（ViT）**：将图像分割为固定大小图块，线性嵌入 + 位置编码，送入标准 Transformer 编码器
- **ViT 变体**：
  - **DeiT**：数据高效，知识蒸馏
  - **Swin Transformer**：分层架构 + 移位窗口，适应高分辨率
  - **LeViT**：分层结构，快速推理
  - **XCiT**：沿通道维度的交叉协方差注意力
  - **MLP‑Mixer**：仅使用 MLP（令牌混合 + 通道混合）

### 5. AbSViT（Analysis‑by‑Synthesis Vision Transformer）
- **灵感**：双稳态感知（鸭子/兔子），大脑做贝叶斯推断，受先验（prior）影响
- **核心思想**：自底向上（bottom‑up）+ 自顶向下（top‑down）的视觉注意力
- **特点**：可用自然语言控制注意力，提升视觉‑语言任务性能
- **机制**：每一层计算受 top‑down signal 调控的视觉注意力，与稀疏重建等价
- **效果**：对同一混合图片，根据不同的类别原型（prototype），注意力集中在相应物体上

### 6. 经典网络架构发展
- CNN 路径：AlexNet → NiN → ResNet → ResNeXt → DenseNet → …
- RNN 路径：RNN → LSTM → GRU → …
- Transformer 路径：Transformer → BERT → ViT → …

### 7. 未来展望
- 计算机视觉任务（尤其是机器人环境理解）推动新型网络结构
- 可能趋势：统一于 Transformer 架构、全连接网络架构，或根据任务选择最佳结构
