## 2. Logistic Regression (Binary Classification)

### Concept
- Logistic regression is a **supervised learning** method for binary classification tasks.
- Despite the name "regression," it is used for **classification** (predicting discrete labels).
- Uses the **sigmoid function** to map outputs to probabilities between 0 and 1.

### Sigmoid Function
$$\sigma(z) = \frac{1}{1 + e^{-z}}$$

### Loss Function
- **Binary Cross-Entropy (BCE)** loss is used:
$$L = -[y \log(\hat{y}) + (1-y) \log(1-\hat{y})]$$

### Training Process
1. Forward pass: compute predictions
2. Compute loss
3. Backward pass: compute gradients
4. Update weights using gradient descent

## 3. Artificial Neurons

### Biological Inspiration
- Modeled after biological neurons in the brain.
- A single artificial neuron: **linear weighted sum of inputs → nonlinear activation → output**.

### Mathematical Formulation
$$y = f\left(\sum_{i} w_i x_i + b\right)$$
Where:
- $x_i$ = inputs, $w_i$ = weights, $b$ = bias
- $f$ = activation function (ReLU, Sigmoid, etc.)

### Activation Functions
| Function | Formula | Properties |
|----------|---------|------------|
| Sigmoid | $\frac{1}{1+e^{-x}}$ | Output in (0,1), used for binary classification |
| ReLU | $\max(0, x)$ | Simple, avoids vanishing gradient |
| Tanh | $\frac{e^x - e^{-x}}{e^x + e^{-x}}$ | Output in (-1,1) |

## 4. Artificial Neural Networks (ANN)

- Single neurons perform **linear classification**, but combinations (multi-layer) handle **complex nonlinear** problems.
- **Multi-layer networks** (deep neural networks) show superior performance in practice.
- Network topology: layers of neurons connected by weighted edges.

## 5. PyTorch Basics

- Recommended for **research** (dynamic graphs, easy debugging); TensorFlow recommended for **deployment**.

### Basic PyTorch Training Loop
```python
import torch
import torch.nn.functional as F

class Model(torch.nn.Module):
    def __init__(self):
        super().__init__()
        self.linear = torch.nn.Linear(1, 1)

    def forward(self, x):
        return F.sigmoid(self.linear(x))

model = Model()
criterion = torch.nn.BCELoss()
optimizer = torch.optim.SGD(model.parameters(), lr=0.01)

for epoch in range(1000):
    y_pred = model(x_data)
    loss = criterion(y_pred, y_data)
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()
```

### Key PyTorch Concepts
- **Tensor**: Multi-dimensional array (similar to NumPy but with GPU support)
- **Autograd**: Automatic differentiation for computing gradients
- **nn.Module**: Base class for all neural network models
- **Training loop**: Forward pass → Loss → Backward pass → Optimizer step
